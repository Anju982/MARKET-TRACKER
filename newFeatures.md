# AI Feature: Multimodal Technical Analyst Dashboard

## 1. Executive Summary

This feature implements a **Multimodal AI Analyst** designed to automate the visual interpretation of financial markets. By integrating **Gemini 2.0 Flash** with a **Streamlit** frontend, the system transforms raw market data into high-fidelity Candlestick visualizations and subsequent actionable intelligence. It replaces traditional rigid algorithmic triggers with flexible, context-aware visual reasoning.

---

## 2. Technical Background & Architecture

### Visual Inference Pipeline

The system operates on a linear pipeline that converts quantitative data into qualitative insights:

1. **Extraction:** Fetching OHLCV (Open, High, Low, Close, Volume) data via `yfinance`.
2. **Transformation:** Calculating technical indicators using `pandas` rolling windows.
3. **Visualization:** Rendering charts using `plotly` to ensure professional-grade visual clarity.
4. **Encoding:** Converting interactive plots to static `.png` via `kaleido` for LLM consumption.
5. **Inference:** Multimodal processing of image + prompt via `google-genai`.

### Indicator Mathematics

The AI interprets price action relative to statistical overlays. For instance, the **Bollinger Bands** used in the analysis are defined by:

$$Middle Band = SMA(Close, n)$$
$$Upper Band = SMA(Close, n) + (k \times \sigma)$$
$$Lower Band = SMA(Close, n) - (k \times \sigma)$$

_Where $n$ is the period, $k$ is the factor of standard deviation ($\sigma$)._

## 4. Feature Specifications

### A. Multimodal Reasoning

Unlike standard RAG, this feature uses **Visual RAG**. The LLM does not just read price values; it analyzes the _geometry_ of the trend lines and the _positioning_ of candlesticks relative to the Bollinger Bands and Moving Averages.

### B. High-Density Prompting

The feature utilizes a "Senior Technical Analyst" persona. The system prompt constrains the AI to:

- Identify trend exhaustion signals.
- Detect volume-price divergence.
- Provide output in a machine-readable JSON format for potential trade execution.

### C. MLOps & Privacy

- **Transient Storage:** Images are stored in `tempfile` and deleted post-inference to ensure data privacy.
- **Latency Optimization:** Gemini 2.0 Flash is utilized to maintain sub-5-second analysis per ticker.

## 5. Usage Logic

1. **User Input:** Enter tickers (e.g., `AAPL, MSFT, BTC-USD`).
2. **Indicator Setup:** Toggle preferred overlays (SMA, EMA, BB).
3. **Analysis:** System generates chart $\rightarrow$ AI reviews $\rightarrow$ Table populates.
4. **Output:** Review the **Overall Summary** for quick decisions or **Individual Tabs** for deep technical justifications.
